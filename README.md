# 🎮 Word Chain Bot

Телеграм-бот для игры в цепочку английских слов.

## 📁 Структура проекта

```
wordbot/
├── bot.py          — главный файл, запускать его
├── config.py       — ВСЕ настройки (токен, айди, таймеры)
├── game.py         — логика игры
├── admin.py        — админ-панель и рассылка
├── word_loader.py  — загрузка слов из файлов
├── database.py     — хранение пользователей и очков
├── words/          — файлы слов по буквам
│   ├── a.txt
│   ├── b.txt
│   └── ...z.txt
├── logs/
│   └── bot.log     — лог всех действий
└── database.json   — база данных (создаётся автоматически)
```

## ⚙️ Установка

### 1. Установи библиотеку
```powershell
pip install python-telegram-bot --upgrade
```

### 2. Настрой config.py

Открой `config.py` и заполни:

```python
BOT_TOKEN = "1234567890:AAF..."   # токен от @BotFather

ADMIN_IDS = [123456789]           # твой Telegram ID (узнать у @userinfobot)

LOG_CHAT_ID = -1001234567890      # ID группы для логов (или None)
```

### 3. Запуск
```powershell
python bot.py
```

## 📖 Правила игры

- Пишешь английское слово (≥3 букв)
- Бот отвечает словом на **последнюю букву** твоего
- Ты отвечаешь на **последнюю букву** слова бота
- Слова **не повторяются**
- **30 секунд** на каждый ход
- За **5 секунд** до конца — предупреждение

## 🏅 Система очков

| Условие | Очки |
|---------|------|
| Каждое слово | +10 |
| Серия 5+ слов | ×2 |
| Серия 10+ слов | ×3 |
| Подсказка | −3 |

---

## 📝 Как добавить слова

Просто открой нужный файл в папке `words/` и добавь слова — **одно слово на строку**:

```
# words/a.txt
apple
arrow
anchor
```

Файл автоматически перечитывается при следующем запуске бота.

## 🛠 Настройки (config.py)

```python
TURN_TIMEOUT    = 30   # секунд на ход
WARNING_TIME    = 5    # предупреждение за N сек
HINT_PENALTY    = 3    # штраф за подсказку
POINTS_PER_WORD = 10   # очков за слово
STREAK_X2       = 5    # серия для ×2
STREAK_X3       = 10   # серия для ×3
```

## 👥 Лог-чат (необязательно)

Чтобы видеть в реальном времени кто что пишет:
1. Создай группу или канал в Telegram
2. Добавь бота туда
3. Узнай ID группы через @userinfobot
4. Вставь ID в `LOG_CHAT_ID` в config.py
