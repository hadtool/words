# 🎮 Word Chain Telegram Bot

Игра в цепочку английских слов с таблицей лидеров, таймером и подсказками.

## ⚙️ Установка

### 1. Установи зависимости
```bash
pip install python-telegram-bot==20.7
```

### 2. Создай бота
1. Открой Telegram и найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя и username боту
4. Скопируй полученный **токен**

### 3. Вставь токен
Открой файл `bot.py` и замени:
```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```
на свой токен:
```python
BOT_TOKEN = "1234567890:AAF..."
```

### 4. Запусти
```bash
python bot.py
```

## 🕹 Команды бота

| Команда | Описание |
|--------|----------|
| `/start` | Главное меню |
| `/start_game` | Начать новую игру |
| `/score` | Посмотреть текущий счёт |
| `/leaderboard` | Таблица лидеров |
| `/rules` | Правила игры |

---

## 🎯 Правила игры

- Ты пишешь **английское слово** (минимум 3 буквы)
- Бот отвечает словом на **последнюю букву** твоего слова
- Ты отвечаешь на **последнюю букву** слова бота
- Слова **не повторяются**
- На каждый ход **30 секунд**

## 🏅 Очки

| Условие | Очки |
|--------|------|
| Каждое слово | +10 |
| Серия 5+ слов | ×2 бонус |
| Серия 10+ слов | ×3 бонус |
| Подсказка | −3 |

## 🔧 Настройки (в bot.py)

```python
TURN_TIMEOUT = 30   # секунд на ход
HINT_PENALTY  = 3   # штраф за подсказку
WIN_STREAK    = 10  # серия для ×3 бонуса
POINTS_PER_WORD = 10
```
